# Audio Files

Place your audio files here to match the filenames in your TSV data.

## Supported Formats:
- WAV (recommended)
- MP3
- M4A
- OGG

## File Structure:
```
audio/
├── 15-15001-13.wav
├── 15-15001-18.wav
├── 15-15001-24.wav
└── ...
```

## Usage:
1. Copy your audio files to this folder
2. Make sure filenames match the `file_name` column in your TSV
3. The application will automatically detect and play audio when available
4. You can also upload audio files manually during correction process
